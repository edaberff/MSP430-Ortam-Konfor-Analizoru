# MSP430 Tabanlı Ortam Konfor ve Stres Analizörü
Bu proje, MSP430G2553 mikrodenetleyici kullanarak ortamın yaşam kalitesini ölçer.

### Özellikler
* **Sensörler:** DHT22 (Sıcaklık/Nem), BH1750 (Işık), Mikrofon (Gürültü).
* **Haberleşme:** Bluetooth (HC-05) üzerinden mobil takip.
* **Algoritma:** Ağırlıklı ortalama ile % üzerinden konfor skoru hesaplama.

### Kullanılan Araçlar
* **Yazılım:** C Dili (Code Composer Studio).
* **Tasarım:** Altium Designer (Şematik & PCB).
